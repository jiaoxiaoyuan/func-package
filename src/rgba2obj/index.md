## rgba2Obj

Demo:

```tsx | pure
import { rgba2obj } from 'func-package';

// 将 rgba 字符串转为 rgba 对象
rgba2obj('rgba(20,100,20,0.2)'); // -> {r: 20, g: 100, b: 20, a: 0.2}
```
