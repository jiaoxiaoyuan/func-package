## genRandomColor

Demo:

```tsx | pure
import { genRandomColor } from 'func-package';

// 获取随机颜色'hsl'|'hsla'|'rgb'|'rgba'|16，默认是rgb格式。
const color = genRandomColor('rgba');
```
