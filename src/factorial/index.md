## factorial(n)

> 阶乘
>
> 参数
>
> n: 需要求阶乘的数字
>
> 返回值
>
> 若参数 n 大于 18，返回值用 bigint 类型表示。

Demo:

```tsx | pure
import { factorial } from 'func-package';

factorial(5); // -> 120
factorial(19); // -> 121645100408832000n
```
