# undefined

`undefined` 属性是一个特殊值。如果已声明了一个变量但还未进行初始化，则其值为 `undefined`。

该属性为 `Global` 对象的一个只读属性(准确地说，是一个常量)。所有主流浏览器均支持该属性。

## 说明

如果一个变量未被初始化赋值，则其值为 `undefined`。如果一个函数没有返回值，则其返回值默认为 `undefined`。
